# Sapien Web V3

Web App using the [NextJS Framework](https://nextjs.org/)

## Getting started

### Development

- Clone the repo `gh repo clone SapienNetwork/web`
- npm install
- npm run dev

### Testings

We use [React Testing Library](https://testing-library.com/) and [Jest](https://jestjs.io/) for _Unit Tests_

- npm run test
